# Onboarding Agent

**Status**: Coming Soon
**Mode**: Voice (Primary)

## Role

Delivers your best welcome pitch as a guided voice tour.

## Capabilities

### 1. Voice-First Walkthrough
- Delivers welcome pitch as voice-guided experience
- Provides step-by-step onboarding
- Interactive voice navigation

### 2. Brand Consistency
- Keeps tone on-brand throughout
- Introduces must-know product moments
- Personalizes based on user segment

## Planned Features

```
New user detected → Start welcome flow
  ↓
Voice greeting (branded)
  ↓
Guided tour of key features
  ↓
Interactive checkpoints
  ↓
Completion celebration
```

## Onboarding Flow Design

### Step 1: Welcome
- Personalized greeting
- Brief value proposition
- Set expectations for tour

### Step 2: Key Features Tour
- Highlight 3-5 core features
- Voice explanation + visual cues
- "Try it now" prompts

### Step 3: Account Setup
- Guide through essential settings
- Voice-assisted form filling
- Progress indicators

### Step 4: First Action
- Encourage first meaningful action
- Celebrate completion
- Offer next steps

## Voice UX Considerations

| Aspect | Approach |
|--------|----------|
| Pacing | Pause for user processing |
| Repetition | Offer to repeat on request |
| Skip | Allow skipping sections |
| Resume | Save progress for later |

## Integration Requirements

- User Profile System
- Feature Flag Service
- Analytics (completion tracking)
- Voice Synthesis Engine
