# Shopping Guide Agent

**Status**: Default (Active)
**Mode**: Voice / Chat

## Role

Presents the perfect talking points for every aisle in seconds.

## Capabilities

### 1. Brand-Safe Greeting
- Greets visitors with personalized, brand-safe intros on every page
- Adapts tone and messaging based on page context
- Maintains consistent brand voice across interactions

### 2. Contextual Recommendations
- Surfaces recommendations using live catalog data
- Integrates trend data for timely suggestions
- Provides personalized product discovery

## Behavior

```
Visitor arrives → Detect page context
  ↓
Generate personalized greeting
  ↓
Surface relevant recommendations
  ↓
Guide through product discovery
```

## Integration Points

- Product Catalog API
- Trend Data Service
- Customer Profile (if available)
- Page Context Detection

## Voice/Chat Differences

| Feature | Voice | Chat |
|---------|-------|------|
| Greeting | Spoken intro | Text with visual |
| Recommendations | Verbal description | Cards/images |
| Navigation | Voice commands | Click/tap |
