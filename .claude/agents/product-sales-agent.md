# Product Sales Agent

**Status**: Active
**Mode**: Voice / Chat

## Role

Keeps buyers in flow with instant specs, sizing tips, and smart add-ons.

## Capabilities

### 1. Deep Product Expertise
- Answers detailed product questions in real-time
- Pulls from specs, inventory, and reviews
- Provides sizing recommendations
- Explains technical specifications clearly

### 2. Smart Upselling
- Suggests comparisons between products
- Recommends accessories and add-ons
- Provides add-to-cart nudges at optimal moments
- Cross-sells complementary items

## Behavior

```
Product inquiry → Retrieve product data
  ↓
Answer with specs/reviews/inventory
  ↓
Identify upsell opportunity
  ↓
Suggest comparison/accessory
  ↓
Nudge toward purchase
```

## Data Sources

- Product Specifications Database
- Real-time Inventory System
- Customer Reviews
- Pricing Information
- Compatibility Data

## Sales Tactics

| Tactic | Trigger | Action |
|--------|---------|--------|
| Comparison | "Which is better?" | Show side-by-side |
| Accessory | After product view | Suggest add-ons |
| Size Help | Size question | Guide to right size |
| Social Proof | Hesitation | Show reviews/ratings |

## Metrics

- Conversion Rate
- Average Order Value
- Add-on Attach Rate
- Question Resolution Rate
