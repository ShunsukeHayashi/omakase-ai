## 2. 直接競合分析

### 2.1 VAPI (Voice AI Platform)

**概要**: omakase.aiが使用している音声AIプラットフォーム

| 項目 | 詳細 |
|-----|------|
| **ポジション** | 汎用音声AI API |
| **強み** | 即座に使える、豊富な統合、500ms未満レイテンシ |
| **弱み** | カスタマイズ限界、コスト高 (スケール時) |
| **料金** | 従量課金 (分単位) |
| **顧客** | 350K+開発者、150M+通話処理 |

**omakase.aiとの関係**:
- 現在のコア依存サービス
- 差別化困難 (他社も同じVAPIを使用可能)

### 2.2 Pipecat (Open Source)

**概要**: Daily.co提供のOSS音声AIフレームワーク

| 項目 | 詳細 |
|-----|------|
| **ポジション** | 開発者向けOSSフレームワーク |
| **強み** | 無料、完全カスタマイズ可能、ベンダー中立 |
| **弱み** | 開発工数必要、インフラ自己管理 |
| **料金** | 無料 (BSD-2ライセンス) |
| **対応** | Claude, GPT, Deepgram, ElevenLabs等 |

**omakase.aiへの示唆**:
- VAPI代替として検討価値
- コスト削減とカスタマイズ自由度向上

```python
# Pipecat構成例
from pipecat.services.anthropic import AnthropicLLMService
from pipecat.services.deepgram import DeepgramSTTService
from pipecat.services.elevenlabs import ElevenLabsTTSService
from pipecat.transports.services.daily import DailyTransport

pipeline = Pipeline([
    DailyTransport(),         # WebRTC (Daily.co)
    DeepgramSTTService(),     # 音声認識
    AnthropicLLMService(),    # Claude LLM
    ElevenLabsTTSService(),   # 音声合成
])
```

### 2.3 Retell AI

**概要**: 電話対話に特化した音声AI

| 項目 | 詳細 |
|-----|------|
| **ポジション** | 電話自動化特化 |
| **強み** | 電話統合、HIPAA準拠、リアルタイム分析 |
| **弱み** | Webウィジェット非対応 |
| **料金** | 従量課金 |

**omakase.aiとの違い**:
- Retell: 電話中心
- omakase.ai: Webウィジェット中心

### 2.4 Bland AI

**概要**: 電話発信自動化

| 項目 | 詳細 |
|-----|------|
| **ポジション** | アウトバウンド電話特化 |
| **強み** | 大量発信、スケーラブル |
| **弱み** | インバウンド/Web非対応 |

### 2.5 Hume AI

**概要**: 感情認識AI

| 項目 | 詳細 |
|-----|------|
| **ポジション** | 感情分析特化 |
| **強み** | 感情認識、表情分析 |
| **ユースケース** | メンタルヘルス、カスタマーサービス |

### 5.3 競争優位確立のための提案

#### 短期 (3-6ヶ月)

1. **プロンプト・シナリオの深化**
   - 業種別テンプレート (アパレル、食品、家電等)
   - 購買心理に基づく会話フロー改善
   - 日本語特化の自然な応答

2. **分析ダッシュボード**
   - 会話分析
   - コンバージョン追跡
   - 離脱ポイント特定

#### 中期 (6-12ヶ月)

1. **Pipecat移行検討**
   - VAPI依存脱却
   - コスト最適化
   - カスタマイズ自由度

2. **データ蓄積・学習**
   - 会話ログからの学習
   - 商品レコメンドアルゴリズム改善
   - 業界特化モデル構築

#### 長期 (1-2年)

1. **自社音声AI基盤**
   - Fine-tunedモデル
   - 独自STT/TTS
   - 差別化技術の内製化

2. **プラットフォーム化**
   - EC事業者向けSDK
   - マーケットプレイス
   - API公開
