# つじり八十八夜 200g

## 商品概要

祇園辻利を代表する煎茶。「八十八夜」の名は、立春から88日目（5月上旬）に摘まれる最も品質の高い新茶に由来。茶葉に茎を少量ブレンドし、自然な甘みとまろやかさを実現。

## 特徴・こだわり

- **由来**: 八十八夜（立春から88日目）摘みの新茶にちなんだ名前
- **ブレンド**: 茶葉に適度な茎を加え、まろやかな味わいに
- **甘み**: 特選より控えめな甘さで、幅広い方に好まれる味
- **用途**: 日常使いから来客用まで幅広く対応

## おすすめの飲み方

1. 急須に茶葉8g（2人分）を入れる
2. 70-80度のお湯を注ぐ
3. 約1分半待つ
4. 最後の一滴まで注ぎ切る
