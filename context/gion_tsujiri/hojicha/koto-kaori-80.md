# ほうじ茶 古都かをり 80g

## 商品概要

祇園辻利で最も香り高いほうじ茶。茶寮でも提供される本格的な味わい。厳選された茶葉と茎の両方を使用し、豊かな焙煎香が特徴。

## 特徴・こだわり

- **最高の香り**: 祇園辻利のほうじ茶で最も香り高い
- **茶寮で提供**: 実店舗の茶寮でも使用される品質
- **素材**: 厳選された茶葉と茎の両方を使用
- **低カフェイン**: 他の緑茶より低カフェインでリラックスに最適

## おすすめの飲み方

1. 急須に茶葉を入れる
2. 熱湯を注ぐ
3. 約1分半待つ
4. 注ぎ切る

**ポイント**: ほうじ茶は熱湯でOK。香りを楽しむため、淹れたてをどうぞ。
