# 水出しほうじ茶ティーバッグ（4g×20袋入）

## 商品概要

水出し専用のほうじ茶ティーバッグ。約2時間で香り高い冷ほうじ茶が完成。4種類の焙煎茶葉をブレンドし、奥深い味わいと澄んだ後味を実現。

## 特徴・こだわり

- **時短**: 約2時間で水出し完成（一晩不要）
- **4種ブレンド**: 赤外線焙煎と直火焙煎の茶葉をブレンド
- **香り**: 水出しでも熱湯で淹れたような香り
- **後味**: 澄んだすっきりとした後味

## おすすめの飲み方

1. 容器にティーバッグを入れる
2. 冷水300ml（冷ました湯冷ましか軟水のミネラルウォーター推奨）を注ぐ
3. 冷蔵庫で約2時間待つ
4. ティーバッグを取り出してサーブ
