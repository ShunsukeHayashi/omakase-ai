# 抹茶入り玄米茶 200g

## 商品概要

炒った玄米の香ばしい香りと煎茶のさっぱりした味わい、そして上質な抹茶のまろやかさをブレンド。鮮やかな緑色と豊かな香りが特徴。

## 特徴・こだわり

- **三位一体**: 玄米の香ばしさ + 煎茶のさっぱり + 抹茶のまろやかさ
- **色合い**: 抹茶を加えることで鮮やかな緑色に
- **香り**: 炒り玄米の香ばしい香り
- **大容量**: 200gで約25杯分とコスパ良好

## おすすめの飲み方

1. 急須に茶葉8g（2人分）を入れる
2. 熱湯を注ぐ
3. 30秒〜1分待つ（短めでOK）
4. 注ぎ切る

**ポイント**: 玄米茶は熱湯でサッと淹れるのがコツ
