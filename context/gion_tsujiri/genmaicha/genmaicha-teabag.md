# 玄米茶ティーバッグ（4g×17袋入）

## 商品概要

香り高い玄米茶を手軽に楽しめるティーバッグ。炒り玄米と緑茶を絶妙にブレンドし、香ばしい香りとさっぱりした味わいを実現。

## 特徴・こだわり

- **手軽さ**: 急須不要、カップで簡単に
- **香り**: 炒り玄米の香ばしい香り
- **味わい**: 玄米と緑茶のバランスの良いブレンド
- **ギフト対応**: のし・包装サービスあり

## おすすめの飲み方

1. カップにティーバッグを入れる
2. 約95度のお湯を注ぐ
3. 約1分半待つ
4. ティーバッグを取り出す
