# 玉露ティーバッグ（3g×17袋入）

## 商品概要

最高級茶「玉露」を手軽に楽しめるティーバッグ。急須不要で、低温のお湯で丁寧に抽出することで、玉露特有の濃厚な旨味を味わえる。

## 特徴・こだわり

- **最高級茶**: 日本茶の最高峰「玉露」を使用
- **手軽さ**: 急須不要で本格玉露
- **旨味**: 低温抽出で甘みと旨味を最大限に
- **ギフト対応**: のし・包装サービスあり

## おすすめの飲み方

1. カップにティーバッグを入れる
2. **50度程度**のぬるめのお湯100mlを注ぐ
3. **約3分**じっくり待つ
4. ティーバッグを取り出す

**重要**: 玉露は低温で淹れることが大切。熱湯を使うと渋みが出てしまいます。
