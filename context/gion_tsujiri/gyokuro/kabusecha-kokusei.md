# かぶせ茶 国清 200g

## 商品概要

玉露のような旨味と煎茶の爽やかさを併せ持つ「かぶせ茶」。収穫前の約10日間、茶葉に覆いをかけて日光を遮ることで、甘み成分「テアニン」を豊富に含む。

## 特徴・こだわり

- **製法**: 収穫前10日間、覆いをかけて栽培
- **味わい**: 玉露の旨味 + 煎茶の爽やかさ
- **テアニン豊富**: 日光を遮ることで甘み成分が増加
- **汎用性**: 煎茶と同じ淹れ方でOK、温度で味が変化

## おすすめの飲み方

1. 急須に茶葉8g（2人分）を入れる
2. 70-80度のお湯を注ぐ
3. 約1分半待つ
4. 最後の一滴まで注ぎ切る

**温度による変化**:
- 低め（60度）: 甘みが強調
- 高め（80度）: 爽やかさが際立つ
